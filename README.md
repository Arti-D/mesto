#Место

------

##О проекте

[Сайт]() представляет собой что-то вроде личного блога, в котором можно делиться с другими пользователями местами, в которых вы побывали (или хотите побывать).

------

##Технологии

* HTML
* CSS (flexbox, grid)
* Адаптивная верстка
* Базовый JavaScript
